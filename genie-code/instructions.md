I work on CDM (Common Data Model) data transformation for ERP audit data using Databricks.

## CDM Mapping Tasks

When I ask about ERP data transformation, CDM mapping, schema analysis, column mapping,
transform preview, data validation, or audit data — use the `@cdm-mapping` skill.
It guides you through the exact 4-phase workflow using the CDM MCP tools.

## General Preferences

- Be concise — show results as tables where applicable
- When proposing column mappings, always use table format: Source Column | CDM Field | Confidence
- Always confirm with me before proceeding to the next phase of a workflow
- If a tool call fails, show the error and suggest a fix clearly
